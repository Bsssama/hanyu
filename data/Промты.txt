Действуй как строгий конвертер данных из CSV в JavaScript-массив. 

Твоя задача — обработать строки таблицы и вернуть валидный JavaScript-код. Не придумывай, не добавляй и не изменяй данные, которых нет в исходном тексте. Используй ТОЛЬКО предоставленные строки.

Правила формирования объектов:
1. Ключи объектов (id, character, pinyin, translation) должны быть БЕЗ кавычек.
2. Поле `id`: формируй строго в формате "hsk1_l[Номер урока]_ch[Трехзначный порядковый номер слова]". Порядковый номер должен инкрементироваться глобально или продолжаться с заданного мной индекса.
3. Поле `character`: строка с иероглифом.
4. Поле `pinyin`: массив с одной строкой, где записан пиньинь.
5. Поле `translation`: массив с одной строкой, где записан перевод на русский язык.

Оберни результат в переменную: ` const hsk1_characters = [ ... ];`

Пример объекта на основе строки 2 таблицы:
  {
    id: "hsk1_l1_ch001",
    character: "再见",
    pinyin: ["zàijiàn"],
    translation: ["До свидания"],
  }
